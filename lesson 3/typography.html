Typography

The font being used for the header, header tag, and footer is Quicksand This is the chosen font for these sections because it is easy to read yet stands out to the viewer.

The font being used for the navigation bar and any links in the site is EB Garamond. This is the font for those text areas because of the contrast it will provide with the headers making them noticeable when moving around the site.

The text being used for the headers and other main text is Arial. This is the font for the main text because sans-serif fonts are easier to read on a computer screen and device.
